Attention

        This technique is experimental! It is not meant to be used on a
        production website.
        
Description

        This demo shows that it is possible to reconstruct a specially crafted 
        jpg image
        with alpha transparency using a SVG Filter. No javascript is required.
        The SVG is wrapped in two div elements and styled so that it behaves like 
        a responsive image.

        
Support

        This technique requires 
        SVG Filter support.
        
        
Known Issues

        
    
          
  • Firefox renders visual artifacts when reconstructing the image. A
