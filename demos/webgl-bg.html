Description

        This demo shows how to use the  
        AlphaJpg-WebGL.js
        library to load a specially crafted 
        jpg image
        and use it for an element's background image. The image is reconstructed using
        WebGL and is returned as a base64 encoded data uri.
        
Support

        This technique requires Javascript, and
        WebGL support.
